# 📝 Corretor de Redação ENEM

Sistema web para correção de redações no estilo ENEM, com marcação de trechos e avaliação por competências.

## 🚀 Funcionalidades

- ✅ Seleção de redações de um banco de dados
- ✅ Marcação visual de trechos com erros
- ✅ Avaliação por 5 competências (0-200 pontos cada)
- ✅ Cálculo automático da nota final (0-1000)
- ✅ Campo para comentários gerais
- ✅ Exportação da correção em JSON
- ✅ Interface responsiva

## 📦 Como usar

1. Clone este repositório
2. Abra o arquivo `index.html` no navegador
3. Adicione suas redações no arquivo `js/redacoes.js`
4. Selecione uma redação e clique em "Carregar Redação"
5. Use o marcador para destacar trechos com problemas
6. Avalie cada competência
7. Salve a correção

## 🎨 Personalização

- **Adicionar redações**: Edite o array `redacoes` em `js/redacoes.js`
- **Cores**: Modifique as variáveis CSS em `css/style.css`
- **Competências**: Ajuste os níveis de pontuação no HTML

## 📱 Compatibilidade

- ✅ Desktop (Chrome, Firefox, Edge, Safari)
- ✅ Tablet (suporte a caneta/touch)
- ✅ Mobile (responsivo)
